# BACK END

This project is example of **backend** for authentication app.

# REQUIREMENT

- NODEJS v 14.0 >

## STACK

- DOCKER (for database)
- EXPRESSJS (nodejs framework)

## how to run ?

```sh
yarn run docker:start
```

or

```sh
docker-compose up -d postgres
```

skip this process if you already have own databse on your computer.
